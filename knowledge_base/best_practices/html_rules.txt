HTML RULES:
1. Всегда используй class, не className (это для React)
2. Цвета должны быть валидными: black, white, #000, #fff
3. Для анимаций используй @keyframes + animation
4. Структура: <!DOCTYPE html> → <html> → <head> → <body>
5. НЕ подключай jQuery для простых анимаций - используй чистый CSS
6. Анимация должна работать автоматически, без кликов
